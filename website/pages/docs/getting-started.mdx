## Installation

Inside your React project directory, install nature UI by running the following:

### Requirement

- Follow this guide to install and configure Tailwindcss
  [https://tailwindcss.com/docs/installation](https://tailwindcss.com/docs/installation)

```bash
npm i @nature-ui/core
```

```bash
yarn add @nature-ui/core
```

> For `create-react-app` installation instructions, check this
> [CRA templates guide](/guides/integrations/with-cra).
